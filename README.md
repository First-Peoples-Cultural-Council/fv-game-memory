A firstvoices.com memory game

### Steps to start developing

#### Step 1 - Install dependencies
```bash
yarn
```

#### Step 2 - Run the dev server
```bash
yarn run develop
```

#### Step 3 - Run in browser

Visit http://localhost:8080/ in your browser
